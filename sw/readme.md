# sw

This folder contains scripts and instructions for building executable binaries targeting the KTH subsystem.

## Vesyla-Suite
The **Vesyla-Suite**, developed at KTH, can be used to generate executable binaries for the subsystem using DRRA-based workflows.

To get started:

1. **Install** the [Vesyla-Suite](https://github.com/silagokth/vesyla).
2. **Create** a working directory (e.g., `work`).
3. **Run** the following command inside your working directory:

```bash
vesyla testcase run -d ../src/
```
